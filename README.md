# OpenWorld Chat Architecture

<pre>
                                                                +---------------+
                                                                |               |
                                                                |     USER      |
                                                                |               |
                             +---------------------------+      +-------+-------+
                             |                           |              |
+-----------------------+    |  The username is sent and |              |
|                       |    |  saved to the DB          |              |
|                       |    |                           |   +----------v---------+
|       USER            <----+---------------------------+---+                    |
|       DATABASE        |                                    |   User Joins Chat  |
|                       +---+-----------------------------+-->                    |
|                       |   |                             |  +----------+---------+
+-----------------------+   |   The server sends back     |             |
                            |   the Chats History         |             |
                            |                             |             |
                            +-----------------------------+     +-------v-------+
                                                                |               |
                                                                |     CHAT      |
                                                                |               |
                                                                +---------------+

</pre>

# Chat Component Architecture

<pre>                                                            +--------------------------+
                                                              |                          |
+---------------------------------------------------+         | Holds all the            |
|                                                   |         | Chat components together |
| Chat Container                            +---------------> |                          |
|                                                   |         +--------------------------+
|  +---------------------------------------------+  |
|  |                                             |  |           +----------------------+
|  | Chat Log                               +------------->     |                      |
|  |                                             |  |           | Contains all the     |
|  | +-----------------------------------------+ |  |           | Chat Messages, from  |
|  | |                                         | |  |           | earlier to Present   |
|  | |  Chat Message                           | |  |           |                      |
|  | |                                         | |  |           +----------------------+
|  | |                                         | |  |
|  | |                                    +------------------> +------------------------+
|  | |                                         | |  |          |                        |
|  | +-----------------------------------------+ |  |          | Contains user relate   |
|  |                                             |  |          | info {Name, Message..} |
|  |                                             |  |          |                        |
|  +---------------------------------------------+  |          +------------------------+
|                                                   |
|  +---------------------------------------------+  |
|  |                                             |  |           +----------------------+
|  |   Chat Composer                      +------------------>  |                      |
|  |                                             |  |           | Gets User input,     |
|  +---------------------------------------------+  |           | Binds User input and |
|                                                   |           | Name together        |
+---------------------------------------------------+           |                      |
                                                            +----------------------+
</pre>
